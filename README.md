# aspnet-signalr2-chat

A simple project to demonstrate using ASP.NET SignalR 2

- Configuration using Owin
- Using Service Hubs
- Using Groups
- Overriding lifecycle events for monitoring
- Creating jQuery Clients, with and without a static proxy
